# yaoxfly-utils

## 介绍

本项目包含项目中常用的方法,包括 uni-app 的一些方法的封装,
有待完善

### 安装教程

```
npm i yaoxfly-utils
```

### 使用说明

1. 在 main.js 文件里

```
import utils from 'yaoxfly-utils'
Vue.use(utils)
```

2. 示范

```
console.log(this.$YxUtils.timeStampToDate(1564808776))
```

输出结果:2019-08-3
